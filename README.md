# Deskripsi

## Sistem yang dibuat adalah Inventory System sederhana menggunakan java yang digunakan untuk mengelola data produk dan stok produk. Sistem ini dapat memungkinkan pengguna untuk memasukan data produk seoerti ID produk, nama produk, stok awal, harga, dan kategori. Dengan fungsi utamanya mengecek produk, menambah stok produk, dan mengurangi stok.

# Program dapat:
- Menampilkan stok produk
- Menambah stok
- Mengurangi stok

 Desain Class

# class1
Class ini menyimpan data produk seperti:
- productId
- productName
- stock
- price
- category

Class ini menggunakan atribut private dan method seperti:
- getStock()
- setStock()
- addStock()
- reduceStock()
